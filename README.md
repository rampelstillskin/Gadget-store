# Магазин селфи-палок
### Static site


## Gulp скрипты:
* **imageOptimize:** Оптимизация картинок;
* **serve:** Минификация, компиляция sass и live-server;
* **build:** Финальная сборка: чистка мусора, оптимизация картинок, компиляция sass и копирование в папку(build);
